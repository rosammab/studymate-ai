# StudyMate AI

## Project Overview

StudyMate AI is an AI-powered learning platform that helps students study more effectively using Artificial Intelligence.

Users can upload notes, generate AI summaries, create quizzes, produce flashcards, chat with their notes, and receive personalized study plans.

---

# Project Goal

The goal of this project is to build a production-ready AI learning platform using Django and modern AI technologies while demonstrating professional software engineering practices.

---

# Main Features

- User Authentication
- Upload Study Notes
- AI Summary Generation
- AI Quiz Generator
- AI Flashcard Generator
- Chat with Notes
- Study Planner
- Dashboard
- User Profile
